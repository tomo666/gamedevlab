# 条件分岐

## 目的

- 条件分岐（if / else）について理解する
- 比較演算子・論理演算子の基本を覚える
- ブロックスコープ `{ }` について簡単な概要を理解する

---

## 条件分岐とは？

ゲームでいうと、次のような「分かれ道」を作る機能です。

- HP が 0 以下なら「ゲームオーバー」
- ボタンが押されたら「ジャンプ」
- カギを持っていたら「扉が開く」

その分かれ道を作るのが `if` 文です。

---

## if の基本

`if(条件)` の **条件が true（正しい）なら**、`{ }` の中が実行されます。

```csharp
int hp = 10;

if(hp <= 0) {
    // ここは hp が 0 以下のときだけ実行される
}
```

ポイント：

- `if(...)` の中は **true / false の結果になる式**を書く
- `{ }` は「まとめて実行するブロック」

---

## if / else（どちらか片方に分岐）

`if` のブロックに続けて `else` を付けると、if の条件が false のときの処理を書けます。`else` にも同様にまとめて実行するブロックを書きます。

```csharp
int hp = 10;
JString result = null;

if(hp <= 0) {
    result = new JString("DEAD");
    VMC.Print(result);
    result.Destroy();
} else {
    result = new JString("ALIVE");
    VMC.Print(result);
    result.Destroy();
}
```

- `hp <= 0` のときは `DEAD`
- それ以外は `ALIVE`

---

## else if（分かれ道を増やす）

条件を複数並べたいときは `else if` を使います。

```csharp
int hp = 10;

if(hp <= 0) {
    // DEAD
} else if(hp < 5) {
    // ピンチ
} else {
    // まだ元気
}
```

条件が複数ある場合は、上から順にチェックします。そして、条件が当てはまったブロックが実行されると、この一連の if/else if/else 分岐はそれ以降の条件式は実行されなくなります。

---

## 比較演算子（くらべる記号）

条件の中でよく使う「くらべる記号」です。

- `==` : 同じ
- `!=` : 違う
- `<`  : より小さい
- `<=` : 以下
- `>`  : より大きい
- `>=` : 以上

例：

```csharp
int a = 3;
int b = 5;

if(a == b) {
    // 同じ
}

if(a != b) {
    // 違う
}

if(a < b) {
    // a は b より小さい
}
```

※ `=` を単体で使用した場合は「代入」を意味します。
`a = 3;` は a に 3 を入れる、という意味で、比較ではありません。

---

## 論理演算子（条件を合体させる）

条件を「そして / または / ひっくり返す」で合体できます。

- `&&` : AND（両方 true なら true）
- `||` : OR（どちらか true なら true）
- `!`  : NOT（true/false を反転）

例：

```csharp
int hp = 10;
bool hasKey = true;

// HP がある かつ カギを持っている
if(hp > 0 && hasKey) {
    // 扉が開く
}

// HP がない または カギがない
if(hp <= 0 || !hasKey) {
    // 開かない
}
```

---

## ブロック `{ }` とスコープについて

`{ }` の中で作った変数は、基本的に **その括弧 `{ }` の中だけで使う**ものです。

```csharp
if(true) {
    int x = 10;
    // ここでは x を使える
}

// ここでは x は使えない（外に出ると消えるイメージ）
```

※ JBScript では「スコープの扱い」に独自のルールがあるので、ここでは雰囲気だけ覚えれば OK。
詳しいルールは次の「スコープ」ページで解説します。

---

## よくある間違い

`=` と `==` を混同しないようにしましょう。

- `a = 3;` は代入
- `a == 3` は比較

---

## まとめ

- `if(条件)` で分かれ道を作る
- `else` / `else if` で分かれ道を増やす
- 比較演算子（==, &lt;=, ...）と論理演算子（&&, ||, !）を覚える
- `{ }` は処理のまとまり（スコープは次のページで詳しく！）
