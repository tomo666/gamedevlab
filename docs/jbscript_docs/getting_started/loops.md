# ループ

## 目的

- `for` / `while` ループの基本を理解する
- `break` / `continue` の使い方を知る

---

## ループとは？

同じ処理を、何回もくり返す仕組みのことです。
たとえば「20回だけ動かす」「ボタンが押されている間ずっと動かす」といった場面で使います。

JBScript では、主に次の2種類を使います。

- `for` ループ（回数が決まっているときに便利）
- `while` ループ（条件が true の間くり返す）

---

## forループ

`for` は「カウンター（数を数える変数）を使って、決まった回数だけくり返す」ループです。

```csharp
for(int i = 0; i < 20; i++) {
    // ここが20回くり返される
}
```

### forの3つのパーツ

`for(①; ②; ③)` の中は、次の3つでできています。

1. **カウンタ変数の初期化**（例：`int i = 0`）
2. **くり返すための条件**（例：`i < 20`）
3. **1回くり返すごとに行う処理**（例：`i++`）

上の例だと、`i` は 0 から始まり、1ずつ増えていき、`i` が 20 になったら止まります。

### よく使う例

0から10までカウントアップ：

```csharp
for(int i = 0; <= 10; i++) {
    // i が 0,1,2...10 と変わっていく
}
```

10から0までカウントダウン：

```csharp
for(int i = 10; i >= 0; i--) {
    // i が 10,9,8...0 と変わっていく
}
```

2つ飛ばしで増やす：

```csharp
for(int i = 0; i < 10; i += 2) {
    // i が 0,2,4,6,8 と変わっていく
}
```

## whileループ

`while` は丸括弧内の「条件が true の間、ずっとくり返す」ループです。

```csharp
while(bActive) {
    // bActiveがtrueの間、くり返される
}
```

### 注意：終わらないループ

`while` の条件がずっと true のままだと、ループが止まりません。
その場合は、ループの中で条件が変わるようにします。

例：カウントして、一定回数で止める

```csharp
int count = 0;
while(count < 60) {
    count++;
}
```

## break と continue

ループの途中で動きを変えたいときに使います。

### break（ループを終了する）

```csharp
for(int i = 0; i < 100; i++) {
    if(i == 10) {
        break; // ここでループを抜ける
    }
}
```

### continue（次の周回へスキップする）

```csharp
for(int i = 0; i < 10; i++) {
    if(i % 2 == 1) {
        continue; // 奇数のときは、下の処理を飛ばす
    }
    // 偶数のときだけここに来る
}
```

---

## 1行だけなら `{ }` を省略できる

`if` と同じように、`for` と `while` も **中身が1行だけ** のときは `{ }` を省略できます。

```csharp
for(int i = 0; i < 5; i++) num[i] = i;
while(num[4] > 0) num[4] = num[4] - 1;
```

ただし、処理が増えそうな場合は最初から `{ }` を付けるのがおすすめです。
