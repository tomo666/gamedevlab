# 演算子

## 目的

- JBScriptで使える演算子の種類を知る
- 計算される順番（優先順位）を理解する

---

## 演算子とは

たとえば次の行では、`+` も `=` も演算子です。

- `+` は「足し算」をする演算子
- `=` は「右の値を左の変数に入れる（代入する）」ための **代入演算子**

```csharp
int a = 3 + 5;
```

---

## JBScriptで使える演算子一覧

JBScriptでは、主に次の演算子が使えます。

### 算術（計算）

- `+` 足し算
- `-` 引き算
- `*` 掛け算
- `/` 割り算
- `%` 余り

お金を足したり、HPを減らしたり、「数を動かす」ときに使います。

```csharp
int damage = 7;
int hp = 20;
hp = hp - damage;
```

### 代入

- `=` 代入
- `+=` / `-=` / `*=` / `/=` / `%=` まとめて計算して代入

「変数に値を入れる」ための演算子です。
スコア、HP、コイン数などを更新するときに使います。

特に大事なのが「ショートカット（省略形）」です。

- `a = a + 3` は「a を 3 増やし、その結果を a に代入する」という意味ですが、
  同じことを `a += 3` と短く書けます。

```csharp
int a = 10;

a = a + 3;  // 13

a += 3;     // さらに 3 増えて 16
```

他にも、次のような省略形があります。

```csharp
hp -= 5;    // hp = hp - 5
speed *= 2; // speed = speed * 2
coins /= 2; // coins = coins / 2
```

```csharp
int coins = 0;
coins += 1;  // coins = coins + 1;
```

### 比較（true / false になる）

- `<` / `<=` / `>` / `>=`
- `==` 等しい
- `!=` 等しくない

「どっちが大きい？」「同じ？」をチェックして、結果を true / false で返します。
HPが0以下ならゲームオーバー、みたいな判定で使います。

```csharp
int hp = 10;
bool isDead = (hp <= 0);
```

### 論理（true / false をつなぐ）

- `&&` かつ（AND）
- `||` または（OR）
- `!` 反転（NOT）

「条件を組み合わせる」ための演算子です。
たとえば「カギを持っていて、ドアの近くにいるなら開ける」みたいに書けます。

```csharp
bool hasKey = true;
bool isNearDoor = false;

if(hasKey && isNearDoor) {
  // ドアを開ける
}

if(!isNearDoor) {
  // まだ近くない
}
```

### ビット演算（上級者向け）

- `&` / `|` / `^`  ビットAND / OR / XOR
- `~` ビット反転

フラグ管理や高速なマスク処理など、少し上級者向けの場面で使います。

例：アイテムを「持っている/いない」のフラグを 1つの数にまとめるイメージです。

```csharp
// 例：ビットの場所（0, 1, 2...）を決める
int FLAG_KEY    = 1 << 0; // 0001
int FLAG_SWORD  = 1 << 1; // 0010
int FLAG_SHIELD = 1 << 2; // 0100

int flags = 0;

// カギを持った（ON）
flags = flags | FLAG_KEY;

// カギを持っている？（チェック）
bool hasKey = ((flags & FLAG_KEY) != 0);

// カギを捨てた（OFF）
flags = flags & (~FLAG_KEY);
```

### シフト（上級者向け）

- `<<` 左シフト
- `>>` 右シフト
- `<<=` / `>>=` シフトして代入

数を「2倍/半分」っぽく動かしたり、ビット列をずらしたいときに使います。

```csharp
int x = 3;
int a = x << 1; // 左に1つずらす（2倍）→ 6
int b = x >> 1; // 右に1つずらす（半分）→ 1

// まとめて代入もできます
x <<= 2;        // x = x << 2（4倍）
```

### インクリメント / デクリメント

- `++` 変数の値を1増やす
- `--` 変数の値を1減らす

`i++`（後置）と `++i`（前置）の両方が使えます。
`--` も同じで、`i--` と `--i` が使えます。

```csharp
int i = 0;

i++;   // 1増える（あとで増える）
++i;   // 1増える（先に増える）

i--;   // 1減る（あとで減る）
--i;   // 1減る（先に減る）
```

※「あとで/先に」の違いは、式の中で使ったときに影響します。
最初は `i++` を「i = i + 1 の短縮形」くらいに覚えておけばOKです。

例：同じ「1増やす」でも、式の中では結果が変わります。

```csharp
int i = 5;
int a = i++; // この命令の中では、a に 5 が入り、この命令の実行が終わったら、i が 6 になる

int j = 5;
int b = ++j; // この命令の中で先に j が 6 になり、b に 6 が入る
```

最初は「式の中で使わない」方が安全です。
`i++;` のように 1行で書く使い方だけ覚えておけばOKです。


なお、注意事項として、JBScriptの配列にはインクリメント・デクリメントを使用することができません。つまり、num[i]++ のような使い方はできません。

### メンバーアクセス

- `.` （ドット）

「このオブジェクトの中の機能（メソッド）を呼ぶ」ための記号です。
たとえば画像オブジェクトの透明度を変える、みたいな操作で使います。

```csharp
BackgroundImage.SetAlpha(1.0f);
```

## 優先順位（どれが先に計算される？）

演算子には「先に計算される順番（優先順位）」があります。
JBScriptは基本的に **C#と同じ感覚** で考えてOKです。

よく使う範囲で、ざっくり **上ほど先に計算** されます。

1. `.`（ドット）、`x++` / `x--`（後置）
2. `!` / `~` / `++x` / `--x` / `+x` / `-x`（単項）
3. `*` / `/` / `%`
4. `+` / `-`
5. `<<` / `>>`
6. `<` / `<=` / `>` / `>=`
7. `==` / `!=`
8. `&` → `^` → `|`
9. `&&`
10. `||`
11. `=` や `+=` など（代入）

迷ったら、カッコ `()` を付けて計算の優先度をはっきりさせるのが安全です。

```csharp
int a = 1 + 2 * 3;      // 1 + (2*3) = 7
int b = (1 + 2) * 3;    // (1+2) * 3 = 9
```
