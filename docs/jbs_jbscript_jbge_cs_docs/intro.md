# Overview

This document provides a guide to use the JBGE (Jelly Bean Game Engine) API using JBScript to directly interact with game engines that implement the JBGE Virtual Machine (JBGE VM).

## Prerequisites

Although JBScript looks familiar with C#, it is not a 100% compatible programming language. Please go through the JBScript essentials document page and learn the differences and limitations before proceeding with using JBGE APIs.
